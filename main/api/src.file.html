property spaghettihoops: Dict[str, str]¶

Spaghetti hoops of this object Has as many entries as the parameter a given at construction. If c is True, the values will have “foo”, otherwise “bar”.

myclass = MyClass(a=2, b={}, c=True)
print(myclass.spaghetti_hoops)
{
    "hoop_0": "foo",
    "hoop_1": "foo",
}
myclass = MyClass(a=2, b={}, c=False)
print(myclass.spaghetti_hoops)
{
    "hoop_0": "bar",

}
